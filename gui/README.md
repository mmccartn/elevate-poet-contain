PoemQuest
=========
A website to query the https://poetrydb.org/.

Requirements
------------
 * [NodeJS](https://nodejs.org/) v20+

Installation
------------
`npm install`

Development
-----------
1. Compile and host locally: `npm run start`
2. Navigate to GUI at: [http://localhost:4200/](http://localhost:4200/)
